A friend asked me to retool his Excel spreadsheet containing data for 10k manufactured items. I normalized the data and moved it into Microsoft Access. He said he'd like to put it online, and that was my start with MySQL. The database now uses six tables to serve out information like:

  • Manufacter's PN and description
  • After-market supplier's PN and description
  • Close and almost-close substitutes
  • Sub-parts
  • Categories and sub-categories
  • Specific traits by sub-category
  • Images
  • Usage documents like manuals and wiring schematics

I hope to rebuild Sparts on Yii. Yii is wonderful -- especially if you tried CodeIgniter first. I've also worked a bit with Pyramid, a Python framework.
